# Качество изображений на главной

Чтобы фото на главной странице выглядели чётко на всех экранах (включая Retina):

1. **Разрешение** — используй исходники не меньше **1200px по ширине** (для вертикального формата — 1200×1800 px и выше).
2. **Форматы** — сохраняй в **AVIF** (приоритет) и **WebP** как fallback. AVIF даёт лучшее сжатие при том же качестве.
3. **Retina (2x)** — для экранов с высокой плотностью пикселей добавь в разметку `srcset` с дескриптором `2x`:
   ```html
   <img
     src="./image.webp"
     srcset="./image.webp 1x, ./image-2x.webp 2x"
     width="640" height="960"
     alt="..."
   />
   ```
   Либо одна картинка 1280×1920 и указать в `srcset` тот же файл с дескриптором `2x` — браузер сам подставит её на Retina.
4. **Имена файлов** — после замены файлов обнови пути в `index.html` и `en.html` (блоки `.hero__media` и `.brand__mark`).

Сейчас в разметке для героя используется класс `hero-img` — к нему в `styles.css` применена лёгкая коррекция (контраст, насыщенность), которая немного улучшает вид при недостаточном качестве исходника.
